# RQ4 Results

RQ4 is organized into two retained sub-experiments.

## Part A: Boundary-Search Sensitivity

Directory: `part_a_boundary_search_sensitivity/`

- `boundary_sensitivity.csv`: compact sensitivity table.
- `boundary_sensitivity.json`: sensitivity table with metadata.
- Boundary-search work outputs are regenerated by `reproduce.md`; they are not
  retained in the release snapshot.

## Part B: Masking-Partition Sensitivity

Directory: `part_b_masking_partition_sensitivity/`

- `partition_sensitivity.csv`: final merged partition-sensitivity table.
- `partition_sensitivity_fine_0p1_to_1p0.csv`: fine sweep from `0.1x` to `1.0x`.
- `partition_sensitivity_fine_0p1_to_1p0.json`: fine sweep with metadata.
- `partition_sensitivity_config.json`: final experiment configuration.
- `partition_sensitivity_summary.json`: short summary and key results.
- Stage 3 representative-region inputs and partition work outputs are
  regenerated by `reproduce.md`; they are not retained in the release snapshot.
